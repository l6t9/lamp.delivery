# lamp.delivery

Personal site for LampDelivery, rebuilt with Astro.

The current version keeps the core routes from the previous site:
- home
- blog
- projects
- other stuff
- now-playing JSON endpoint

## Scripts

- `pnpm dev` - start the Astro dev server
- `pnpm build` - typecheck and build for production
- `pnpm preview` - preview the production build
